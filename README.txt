boy.py에 AutoRun 클래스를 구현했습니다.


1. Idle 상태에서 a키를 누르면 AutoRun 상태가 됩니다.

2. boy는 Idle 상태일 때 100*100사이즈, AutoRun 상태일 때 150*150사이즈가 되어 10의 속도로 화면을 이동하며 경계면에서 이동 방향을 전환합니다.

3. AutoRun 상태에서 5초가 되면 Idle상태로 돌아갑니다.

4. AutoRun 상태에서 방향키를 누르면 AutoRun 상태에서 빠져나와 눌렸던 방향대로 5의 속도로 움직이는 Run 상태가 됩니다.


과제 설명에서 Run 상태에서 상호작용을 하는 것을 설명하지 않아
Run 상태에서 아무런 이벤트도 받지 않게 구현했습니다.
